# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-28

### Initial Public Release

- Initial release of course-finder.js for DiscGolfAPI course finder widgets
- Standalone, embeddable disc golf course finder widget
- Support for region, country, and global course searches
- 7-day weather forecasts with Open-Meteo integration
- Course badges for partner and featured courses
- Themeable UI with full customization
- Responsive design with mobile support
- WCAG accessible
- Zero dependencies (~15KB JavaScript)

### Features

- Live course data from DiscGolfAPI
- GB presets (England, Scotland, Wales) + US state support + global search
- Locality search with nearby course discovery
- Weather forecasts (0-7 days configurable)
- Course badges for partner courses and highlights
- Themeable — customize colors to match your brand
- Responsive — works on mobile, tablet, desktop
- Lightweight — ~15KB JavaScript, zero dependencies
- Accessible — WCAG compliant with proper ARIA labels
- Embed-friendly — iframe or inline embed support
- Attribution included — visible DiscGolfAPI credit

### Files

- `assets/course-finder.js` - Main widget JavaScript
- `assets/course-finder.css` - Widget styling
- `assets/course-badges.json` - Course badge data
- `index.html` - Demo and markup reference
- `configurator.html` - Interactive embed code generator

### Documentation

- Comprehensive README with installation instructions
- CONTRIBUTING guide for community contributions
- SECURITY policy for vulnerability reporting
- GitHub issue templates for bug reports and feature requests
